# SpartaDungeon-Game
스파르타 던전 미니 게임

<br><br>

## 💻 프로젝트 소개
C#을 활용해 만든 텍스트 RPG 게임입니다.

<br><br>

## 🕑 개발 기간
• 23.08.17 ~ 23.08.23

### ⚙️ 개발 환경 (필수 요구 사항)
• ```C# - v1.26.0```


• ```.NET - 7.0```


• ```Text Editor - Visual Studio Code```




## 🔫 구현 기능
#### 필수 요구 사항
- 게임 시작 화면
- 상태보기
- 인벤토리


#### 선택 요구 사항
- 아이템 정보를 클래스 / 구조체로 활용해 보기
- 아이템 정보를 배열로 관리하기
- 아이템 추가하기 - 인벤토리에 나만의 새로운 아이템을 추가해보기
- 콘솔 꾸미기 -  콘솔의 색 지정, 라인 정렬등을 이용해 꾸며보기
- 인벤토리 크기 맞춤
- 상점 - 아이템 구매
